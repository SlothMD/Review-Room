# Release Checklist

- Run the validation commands in `refs/testing/validationCommands.yaml`.
- Run the manual extension smoke test on a current Amazon product page.
- Confirm `manifest.json` version and permissions are intentional.
- Confirm icons are present at all declared sizes.
- Confirm README setup steps match current behavior.
- MVP release/distribution target is an unpacked Chrome extension folder.
- Browser store packaging and non-Chrome builds are out of MVP scope.
